## Investing for Programmers

- **作者**：Stefan Papp（20 年投资经验 + 数十年数据工程/软件顾问）
- **出版社**：Manning Publications
- **ISBN**：9781633435803

### 书的结构

| 章节 | 内容 |
|------|------|
| 1+12 | 资产类型：股票/债券/外汇/加密货币/ETF/对冲基金 |
| 2 | 金融数据与非金融数据（技术面、基本面、情绪、ESG） |
| 3 | 平台接入（Yahoo Finance、Finviz 等 API） |
| 4+5 | 投资策略：收入型/价值型/成长型组合 |
| 6 | 资产监控（净值、盈亏） |
| 7 | 风险管理（对冲、压力测试、组合优化） |
| 8 | 机器学习模型推理 |
| 9 | GenAI：AI Agent + LLM + MCP + RAG 做投资决策 |
| 10 | 图表分析（均线、布林带、一目均衡表） |
| 11 | 算法交易（买/卖/持有信号） |

### 时效性

书中包含 **MCP（Model Context Protocol）**章节，说明内容非常新（2025-2026年出版）。核心是方法论和系统架构，时效性长。
