注意这个要么跟随，要么自成一体的思想，在连续数组中很常见
```java
     public int maxSubArray(int[] nums) {
	     int n = nums.length;
	     if(n==0) return 0;
	     int last = nums[0];
	     int cur = 0;int result = last;
	     for(int i=1;i<n;i++) {
	     // 要么跟随，要么自成一体
		     cur = Math.max(nums[i], nums[i]+ last);
		     last = cur;
		     result = Math.max(result, cur);
	     }
	     return result;
     }
```
