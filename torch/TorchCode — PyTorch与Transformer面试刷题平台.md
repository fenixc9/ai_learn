TorchCode  一个专门练习 PyTorch 核心实现与 Transformer 面试题的自托管刷题平台。

  → 把 ML 面试中常见的 PyTorch 算子、模块和架构实现整理成 39 道结构化练习题。
  → 提供自动判题、梯度校验、计时和即时反馈，让练习过程更像面试版 LeetCode。
  → 基于 Jupyter Notebook 构建，同时支持一键重置、提示、参考解和进度追踪。
  → 覆盖 ReLU、Softmax、LayerNorm、Attention、RoPE、Flash Attention、LoRA、MoE 等高频主题。
  → 支持 Hugging Face Spaces 在线体验、Google Colab 单题打开，以及 Docker 本地运行。
  → 不需要 GPU，适合准备 ML Engineer、LLM Engineer 和相关研究岗面试的人。
